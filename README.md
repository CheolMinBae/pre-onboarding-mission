### 기능
- input - focus시 outline 색 다르게 처리
- 검색 리스트에 type(ex. COMPANY) 추가
- 검색 키워드에 해당되는 문자열 볼드 처리
- throttling 적용
- 검색 리스트 hover시 해당 항목 css 바뀌게 처리
- 컴포넌트 unmount 및 검색키워드 변경할때마다 `clearTimeout`을 통해 예상치 못한 타이머 동작 방지
